# 📚 오픈소스소프트웨어(OSS) 학습 정리

이 저장소는 오픈소스소프트웨어 교과목을 수강하며 **Git과 GitHub의 핵심 개념 및 명령어**를 정리하기 위해 생성됨

## 1. Git & GitHub란?

| 구분 | 설명 |
| :---: | --- |
| **Git** | 소스 코드의 변경 이력을 관리하는 **분산 버전 관리 시스템(VCS)**입니다. 인터넷 없이 로컬에서도 작업이 가능합니다. |
| **GitHub** | Git 저장소를 온라인상에서 호스팅해주는 **웹 서비스**입니다. 협업, 코드 공유, 포트폴리오 관리에 사용됩니다. |


## 2. 필수 명령어 정리 (CLI)

실무와 오픈소스 프로젝트에서 자주 사용하는 명령어들을 카테고리별로 정리했습니다.

### 🚩 1. 초기 설정 및 조회 (Config & Check)
| 명령어 | 설명 |
| --- | --- |
| `git config --global user.name "이름"` | 사용자 이름 설정 (커밋 기록용) |
| `git config --global user.email "메일"` | 사용자 이메일 설정 (GitHub 계정과 동일하게) |
| `git status` | 현재 파일들의 상태(변경, 스테이징 등) 확인 |
| `git log` | 현재까지의 커밋 기록(History)을 자세히 조회 |
| `git log --oneline` | 커밋 기록을 한 줄씩 간략하게 조회 |
